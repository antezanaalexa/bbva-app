## 🧭 Orden Sugerido para Revisar la Entrega

1. **Paso 1:** Leer el **Resumen Ejecutivo (01)** para entender el valor del proyecto.
2. **Paso 2:** Analizar el **Tarifario y Tasa (02)** para validar los cálculos financieros.
3. **Paso 3:** Revisar los **Requisitos del Crédito (03)** y las evidencias sugeridas.
4. **Paso 4:** Seguir el **Manual de Power BI (10)** para conectar las vistas SQL de la base de datos al dashboard.
5. **Paso 5:** Seguir los **Casos de Prueba (05)** para comprobar el funcionamiento lógico e integrado de extremo a extremo.
6. **Paso 6:** Validar la seguridad técnica en el **Reporte de Ciberseguridad (06)**.
